# ken-all-mcp

日本郵便の郵便番号データ（KEN_ALL.CSV）を使用して、住所や郵便番号を検索できるMCPサーバーです。

## 機能

- 住所や郵便番号の検索
- 複数キーワードでの検索（スペース区切り）
- 最大10件までの検索結果表示

## 使用方法

1. サーバーの起動

```bash
deno run --allow-read server.ts
```

2. 検索の実行

以下のような形式で検索を実行できます：

```json
{
  "keywords": "検索キーワード"
}
```

### 検索例

- 都道府県名での検索：
```json
{ "keywords": "北海道" }
```

- 市区町村名での検索：
```json
{ "keywords": "札幌市 中央区" }
```

- 郵便番号での検索：
```json
{ "keywords": "0600000" }
```

- 町域名での検索：
```json
{ "keywords": "大通" }
```

## 検索結果の形式

検索結果は以下の形式で返されます：

```
郵便番号, 都道府県名, 市区町村名, 町域名
```

例：
```
0600000, 北海道, 札幌市中央区, 以下に掲載がない場合
0600041, 北海道, 札幌市中央区, 大通東
```

## 必要なファイル

- `utf_ken_all.csv`: 日本郵便の郵便番号データ（KEN_ALL.CSV）
- `server.ts`: MCPサーバーのソースコード

## 注意事項

- 検索結果は最大10件まで表示されます
- 検索は部分一致で行われます
- 大文字小文字は区別されません
